  • Zufall im Browser

 ×  Was? Mit dieser Webseite kann man Zufallsexperimente beschreiben, simulieren und analysieren. Sie wurde für den Mathezirkel Augsburg erstellt.

Wie? Im linken Teil notiert man Arbeitsanweisungen an den Computer in der Programmiersprache Python. Dabei kann man einige vordefinierte Befehle und Variablen verwenden, die für die Beschreibung von Zufallsexperimenten zweckmäßig sind. Diese werden weiter unten erklärt. Ein Klick auf „Los!“ in der Menüleiste führt dazu, dass das Programm wiederholt ausgeführt wird. Der Endstand aller vorkommenden Variablen in jedem Durchgang der Simulation wird in Histogrammform im rechten Teil des Fensters angezeigt.

Fragen? Bei Fragen einfach dem Autor eine E-Mail schreiben.

Über diese Umsetzung: Der eigene Python-Code verlässt den Browser nie und auch die statistische Auswertung erfolgt vollständig im Browser. Diese Webseite funktioniert daher auch offline, wenn man sie lokal speichert. Außerdem muss man selbst bei lang laufenden Zufallexperimenten keine Sorgen haben, einen zentralen Server zu überlasten. Dieses Projekt baut auf den Open-Source-Projekten Skulpt, CodeMirror und D3.js auf. Pull Requests sind willkommen!
