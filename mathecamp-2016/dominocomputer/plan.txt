=== AND-Schaltung

  +-------------+--->
  |l            |
  |            s|       Längen: l >= r + s
  |        r    |
  +---   -------+       tb+ < ta- + r
      \ /     |
       |      |
       |      |
       a      b

Damit diese Schaltung funktioniert, müssen der Umweg (hoch rechts) und
die Verbindung von a-Verzweigung zu b-Endpunkt hinreichend lang sein.

=== Rollende Kugeln

Man kann auch einen Computer aus rollenden Kugeln bauen!

* Entweder in kompletter Analogie zum Dominocomputer. Dazu benötigt man eine
  Art Schalter, der das vorzeitige Stoppen bei Dominosteilen emuliert.

* Oder wie bei Kryptographie mit Spielkarten. Dazu benötigt man
  eine Möglichkeit, eine Bahn bei Eintreffen einer Kugel umzuschalten.
